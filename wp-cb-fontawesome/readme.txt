Wp Cb FontAwesome helps bloggers to migrate from FontAwesome 4 to 5 withoud loosing previous icons
 
== Description ==
 
=ENG=

Wp Cb FontAwesome is a plugin to migrate from [FontAwesome](http://www.fontawesome.com "FontAwesome") 4 to 5 in the easiest way possible.
Plugin works in background, you don't need to do more than activate the plugins. It will add required libraries in the header of your wordpress installation.
Wp CB Fontawesome also add the "defer tag" automatically as required from FontAwesome.
All contents are in italian, provided by [CreareBlogs.net](http://www.creareblogs.net "CreareBlogs.net"), many thanks to [WpDeal.net](http://www.wpdeal.net, "WpDeal.net") for the english translation.

* Plugin's page: [CreareBlogs.net](http://www.creareblogs.net/fontawesome-su-wordpress/ "Installare FontAwesome su Wordpress - CreareBlogs.net")

After installation navigate to [FontAwesome Icon Page](https://fontawesome.com/icons?d=gallery "FontAwesome icons page") to use the icons

=IT=

Wp Cb FontAwesome è il miglior plugin per wordpress che ti consente di migrare dalla versione 4 alla versione 5 di FontAwesome](http://www.fontawesome.com "FontAwesome") nella maniera più facile possibile.
Il plugin lavora in background, e per utilizzarlo ti basterà attivarlo dal pannello dei plugin di wordpress.
Wp Cb FontAwesome aggiunge anche il tag "Defer" come richiesto da Fontawesome.

== Installation ==
 
To install Wp Cb FontAwesome you can:

1. Join your wordpress Dashboard;
2. Search for "wp cb fontawesome" in the plugins search box;
3. Install and Activate;

or

1. Download the plugin;
2. Extract the zip file;
3. Upload via FTP into "wp-content/plugins" folder;
4. Activate via Dashboard.
